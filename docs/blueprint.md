# **App Name**: HarmonyStream

## Core Features:

- Streaming Service Integration: Connect to multiple streaming services (e.g., YouTube Music) via their APIs.
- Unified Search: Search across all integrated streaming services simultaneously.
- Audio Playback: Play audio streams directly within the web app.
- Playlist Management: Create, manage, and save playlists of tracks from different services in the Firestore database.
- Personalized Recommendations: Generate personalized music recommendations based on listening history from streaming services by selecting and ordering search queries and content for the user. The recommendations tool adapts to your preferences.

## Style Guidelines:

- Primary color: Soft indigo (#7986CB) for a calm and immersive feel.
- Background color: Dark gray (#303030) for a modern, distraction-free experience.
- Accent color: Electric purple (#BA68C8) for highlights and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, readable interface.
- Minimalist line icons for controls and navigation.
- Clean and intuitive layout with a focus on album art and track information.
- Subtle transitions and animations for a smooth user experience.
